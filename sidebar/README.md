### 使用 Usage

1. 安装 Install

  ```sh
  $ bower install --save-dev brix-component-sidebar
  ```

2. 配置组件 Package

  ```js
  require.config({
    paths: {
      'component/sidebar': 'bower_components/brix-component-sidebar/sidebar'
    }
  })
  ```

3. 应用组件 Apply

  ```html
  <div bx-id="component/sidebar" bx-options="{}"></div>
  ```
